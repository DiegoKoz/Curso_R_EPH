# Curso_R_EPH

Curso de R para procesamiento de datos de la Encuesta Permanente de Hogares

## syllabus 

### Clase 1:

Introducción, lógica sintáctica, comandos elementales y lectura/escritura de archivos.

### Clase 2:

Manipulación de Bases de Datos. Ejercicio práctico: Replicar principales cuadros del Informe Técnico de Mercado de Trabajo

### Clase 3:

Gráficos. Caso práctico: Replicar gráficos del Informe Técnico de Distribución del Ingreso. Armado de gráficos más laborados a partir 
de la utilización de múltiples variables.

### Clase 4:

Ejercicio Integrador: Trabajo con Datos de panel para observar transiciones individuales entre las situaciones de Indigencia-Pobreza-No Pobreza. Gráficos de transición.

### Clase 5:

Regresión Lineal y datos de Panel
